## 🎯 学习建议（初学者路径）

### 第一步：选择工具
```bash
# 推荐3个中最简单的一个开始：
1. Bolt.new - 如果你想快速看到结果
2. Replit - 如果你想学习编程基础
3. v0.dev - 如果你想专注前端
```

### 第二步：基础概念学习
- 花1-2小时了解"什么是AI编程"
- 学习如何给AI写清晰的指令
- 理解AI能做什么和不能做什么

### 第三步：实际练习
- 从简单项目开始（如待办事项列表）
- 逐步增加复杂度
- 记录遇到的问题和解决方法

### 第四步：进阶工具
- 熟练后可以尝试Cursor或Windsurf
- 学习编写简单的rules文件
- 探索更高级的功能

## ⚠️ 初学者注意事项

1. **不要过度依赖AI** - 理解生成的代码
2. **从简单项目开始** - 避免一开始就做复杂应用
3. **多练习prompt编写** - 这是AI编程的核心技能
4. **保持耐心** - AI会犯错，需要反复调试
